Come leggere il cruscotto: `Conteggio` = numero di negozi, articoli o righe importate. `Paia` = quantità fisiche suggerite, trasferite o ordinate. `€` = valore economico stimato degli ordini. `Sellout` = percentuale media di rotazione merce. `Mancanza stimata` = domanda stimata maggiore dello stock dopo i movimenti. `Confronto vs aggiornamento precedente` = differenza rispetto alla run DB immediatamente prima. `25i/25e` = stagioni correnti, `25y/25g` = continuative. `base`, `storico`, `ibrido` = i tre metodi di previsione ordini.
