Come leggere il cruscotto: `Conteggio` = numero di negozi, articoli o righe importate. `Paia` = quantità fisiche suggerite, trasferite o ordinate. `€` = valore economico stimato degli ordini. `Sellout` = percentuale media di rotazione merce. `Mancanza stimata` = domanda stimata maggiore dello stock dopo i movimenti. `Confronto vs aggiornamento precedente` = differenza rispetto alla run DB immediatamente prima. Sigle stagioni: `I/Y` = inverno, `E/G` = estate. `Y` e `G` indicano i continuativi. `base`, `storico`, `ibrido` = i tre metodi di previsione ordini.

Catalogo Stagionale

Importa i file esportati dal gestionale direttamente nel database BARCA. Gli Excel `.xls/.xlsx` vengono letti come catalogo, i `.csv` vengono riconosciuti automaticamente come `listino` o `saldo` dal contenuto, anche se il nome file non viene cambiato. La consultazione usa sempre l'ultimo valore importato per stagione, articolo e negozio, mantenendo lo storico dei run.
